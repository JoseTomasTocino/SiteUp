Plataforma web

Gestión de chequeos

  • Organización por grupos
  • Activación/desactivación autónoma
  • Cuatro tipos distintos
  • Título y descripción
  • Granularidad máxima: 1 chequeo por minuto
  • Notificaciones: correo electrónico y aplicación Android

Plataforma web

Ping check

  • Chequeo de servidores por paquetes ICMP
  • Posibilidad de poner un tope de tiempo de respuesta máximo
  • Gráficas y estadísticas según tiempo de respuesta

Plataforma web

Port check

  • Chequeo remoto de puertos
  • Se abre una conexión al servidor remoto en el puerto indicado
  • Posibilidad de revisar la respuesta textual del servidor

Plataforma web

DNS check

  • Chequeo de registros DNS
  • Tipos a revisar:
    • A: address record
    • AAAA: IPv6 address record
    • CNAME: canonical name (alias)
    • MX: mail exchange
    • TXT: text record
  • Debe coincidir algún registro con el contenido indicado

Plataforma web

HTTP check

  • Chequeo por peticiones HTTP
  • Revisión del código de estado:
    • 200: OK
    • 404: Not found
    • 301: Moved permanently
    • 500: Internal Server Error
  • Opcionalmente, revisión de contenido
